# Guardian of Galaxy

A simple 2D space shooter game made with **Godot 4.6**.

## Description
- Pilot your spaceship and fight enemies across the galaxy.
- Collect resources, upgrade your ship, and defeat bosses.
- Developed using Godot Engine 4.6
